Redes de Internet

Un poco del conocimiento que quiero compartir

Proveedor de Servicios de Internet Inalámbrico (WSIP)

Mi trabajo acerca del conocimiento sobre redes de internet se puede definir como WSIP, que son las siglas en inglés de Proveedor de Servicios de Internet Inalámbricos. En este apartado, daré más detalles sobre en qué consiste.

Conociendo la Zona

En primer lugar, es fundamental conocer la zona en donde vas a proveer el servicio, porque depende del tipo de antenas que estén como punto de acceso y la cobertura o línea de vista que poseen. Un ejemplo para entender esto es imaginarnos que es de noche y tienes una lámpara; al encenderla, dependiendo del tipo de lámpara, es el alcance que logrará iluminar y también la cobertura. Muy similar ocurre con las antenas, con la diferencia de que entran en juego detalles técnicos como el tipo de antena (lineales, de panel y omnidireccional), la frecuencia de trabajo y el factor de ruido.

Si bien ya hablé de la línea de vista, se debe verificar que no existan obstáculos como árboles, construcciones o cualquier otra barrera física que pueda interferir con la señal.

Configuración del Enlace

Este apartado comprende de forma general dos partes: la configuración del enlace (antena en el lugar de instalación y antena base) y la administración de los clientes en cuanto a las direcciones IP proporcionadas por el proveedor principal y la IP de administración.

Configuración del Router Wifi

Normalmente se comienza con la configuración del router wifi, el cual suele tener una dirección IP por defecto como 192.168.0.1. Esta dirección no necesita ser modificada a menos que ya esté en uso. Aquí se establece el nombre de la red wifi y la contraseña.

Configuración de la Antena

La antena exterior en el domicilio debe apuntar a la antena base para generar el enlace. Se trabaja para encontrar la señal más óptima y vincular la dirección IP para su administración. Es importante usar la IP correcta para evitar problemas de conexión y poder acceder remotamente.

Administración de Direcciones IP

Este apartado se divide en dos puntos: la dirección IP del proveedor y la IP de administración.

Dirección IP del Proveedor

La dirección IP del proveedor debe ser única y no debe ser usada por ninguna antena o equipo de administración para evitar conflictos de comunicación. Además, se consideran las DNS, que son como traductores relacionados con las direcciones IP.

IP de Administración

En la IP de administración, comúnmente se usa un RB Mikrotik para gestionar las conexiones de las antenas base y las estaciones de clientes. Es importante tener IPs únicas y evitar repetir segmentos para mantener una administración eficiente.

Se que hay más complejidad en el contexto de la conexión, pero de manera general es como se llevan a cabo los enlaces WISP.
